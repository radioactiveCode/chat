1) Uruchom server MySQL w wersji 8.0.22 (użytkownik: root, hasło: login, port: 3306)
2) Zaloguj się, a następnie utwórz bazę danych o nazwie chat (schema.sql) oraz 2 tabele:
- uzytkownicy (uzytkownicy.sql)
- logowania (logowania.sql)
3) Utwórz konto Administratora (admin.sql)
4) Uruchom server PHP w wersji 7.3.28 (XAMPP Control Panel)
5) W lokalizacji C:\xampp\htdocs\ utwórz katalog o nazwie chat
6) Przekopiuj pliki projektu (4 x CSS, 4 x JS, 14 x PHP, 1 x PNG) do stworzonego katalogu
7) Wyświetl okno logowania wpisując do przeglądarki http://localhost/chat/index.php
8) Zaloguj się jako Administrator, a następnie postępuj według Instrukcji Użytkownika
